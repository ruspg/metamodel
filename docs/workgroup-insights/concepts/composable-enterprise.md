# Composable Enterprise

## Определение
Модель организации банка, в которой value streams собираются из переиспользуемых capability-кубиков. Устоявшийся рыночный термин, предложенный рабочей группой (2025-12-01) вместо «модульный банк».

## Связанные entity_kinds
- `business_capability` — переиспользуемые кубики
- `value_stream` — сквозные потоки, собираемые из capabilities
- `bank_product` — продукт, реализуемый через value streams

## Суть идеи
При запуске нового продукта:
1. Смотрим какие capabilities уже есть
2. Переиспользуем существующие
3. Достраиваем недостающие
4. Собираем value stream из кубиков

**Ключевой use-case:** быстрый time-to-market через понимание существующих capabilities.

## Источник
- Встреча 2025-12-01, предложение рабочей группы
- Рабочая группа согласилась заменить «модульный банк» на composable enterprise
