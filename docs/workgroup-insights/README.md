# Insights рабочей группы по метамодели

Результаты анализа 55 транскрипций встреч (декабрь 2025 -- март 2026).

## Содержание

| Файл | Описание |
|------|----------|
| [dashboard.html](dashboard.html) | Интерактивный дашборд: концепция + статистика WG |
| [summary.md](summary.md) | Сводное понимание концепции метамодели |
| [evolution.md](evolution.md) | Хронология эволюции идей |
| [decisions.yaml](decisions.yaml) | Ключевые решения (YAML-формат) |
| [research_brief.md](research_brief.md) | Задание на рисерч рыночных практик |
| [concepts/](concepts/) | Справочник концепций (8 тем) |

## Как пользоваться

- **dashboard.html** -- откройте в браузере, single-file без зависимостей
- **summary.md** -- начните с него для общей картины
- **concepts/** -- углублённое описание по каждой концепции

## Источник

Сгенерировано на основе LLM-анализа транскрипций встреч рабочей группы.
Имена участников деперсонализированы; используются роли.
